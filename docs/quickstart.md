# Quickstart

After [installation](installation.md), start with a simple import check:

```python
import pydpeet
```

If that works, the next recommended step is to open the [example](examples/index.md) notebooks or scripts with the interpreter of your active environment and run them from there.


If you are setting up PyDPEET for development rather than only for use, continue with the [developer guide](developer.md).
